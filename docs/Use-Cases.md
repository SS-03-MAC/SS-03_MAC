# Use Cases
## Actors
- Developers
- Web masters
- Users
- Admins

## Functionality
### Developers
- Generate template code from models
- Configure aspects of web/MAC server

### Web masters
- Configure server
- Control server

### Users
- CRUD
  - JSON endpoints
  - JavaScript Functions
  - UI features created by the developers
- ACLs
  - Per field Access controls

### Admins
- CRUD
  - JSON endpoints
  - JavaScript Functions
- ACLs
  - Per field Access controls
  - Admin should be read/edit all fields
